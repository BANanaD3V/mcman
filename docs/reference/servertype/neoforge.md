# NeoForge

Downloads the server jar from [NeoForge](https://neoforged.net/)

**Fields:**

| Name     | Type              | Description           |
| -------- | ----------------- | --------------------- |
| `type`   | `"neoforge"`      |                       |
| `loader` | string/`"latest"` | Loader version to use |

```toml
type = "neoforge"
loader = "latest"
```

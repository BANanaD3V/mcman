# PurpurMC

Downloads the server jar from [PurpurMC](https://purpurmc.org/).

!!! example
    ```toml
    type = "purpur"
    build = "10" #(1)!
    ```

    1. Defaults to `latest`

**Fields:**

| Name    | Type              | Description                    |
| ------- | ----------------- | ------------------------------ |
| `type`  | `"purpur"`        |                                |
| `build` | string/`"latest"` | Optionally define the build id |

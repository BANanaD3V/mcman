# Fabric

Downloads the server jar from [Fabric](https://fabricmc.net/)

**Fields:**

| Name | Type | Description |
| --- | --- | --- |
| `type` | `"fabric"` | |
| `installer` | string/`"latest"` | Installer version to use
| `loader` | string/`"latest"` | Loader version to use

```toml
type = "fabric"
installer = "latest"
loader = "latest"
```
